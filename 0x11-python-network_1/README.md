Python Network Scripts

This repository contains a collection of Python scripts for interacting with web servers using various network protocols.

Scripts Description

0-hbtn_status.py: Fetches the status of a given URL using urllib.
1-hbtn_header.py: Retrieves the value of the X-Request-Id variable from the header of a URL response using urllib.
2-post_email.py: Sends a POST request to a URL with an email as a parameter using urllib.
3-error_code.py: Displays the body of a URL response and handles HTTP errors using urllib.
4-hbtn_status.py: Fetches the status of a URL using the requests package.
5-hbtn_header.py: Retrieves the value of the X-Request-Id variable from the header of a URL response using requests.
6-post_email.py: Sends a POST request to a URL with an email as a parameter using requests.
7-error_code.py: Displays the body of a URL response and handles HTTP errors using requests.
8-json_api.py: Sends a POST request with a letter parameter to search for a user using requests.
10-my_github.py: Uses the GitHub API with Basic Authentication to display user information.
Requirements

Python 3.8.5
Ubuntu 20.04 LTS
pycodestyle 2.8.* for code style checking
Usage

Each script can be executed from the command line by running ./script_name.py.
